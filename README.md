# Chatbot_AI
Criação de um chatbot utilizando python como liguagem e Gemini como IA gereativa para geração de conversas com o chat

Primeiro passo necessário gerar uma Chave API para utilização da IA
Baixar as dependência da IA generativa
Após isso seguir com a importaçao da biblioteca e comandos para executação do chat
Finalizando com a gravação dos registros do chat em um histórico.
